বঙ্গাব্দ (Bongabdo) - Modern Bengali Calendar

Today's Date (Default Format)

bongabdo() // Returns today's date in default format

Specific Date (2024-01-15)

bongabdo('2024-01-15') // Convert specific date

With Season

bongabdo('2024-01-15', { format: 'DD MM, YY (SS)', showSeason: true })

With Weekday

bongabdo('2024-01-15', { format: 'DD MM, YY (WW)', showWeekDays: true })

Custom Format

bongabdo('2024-01-15', { format: 'DD-MM-YY' })

Raw Date Object

getBanglaDate('2024-01-15') // Returns raw date object

Format Examples

Only Month

format: 'MM'

Only Year

format: 'YY'

Only Season

format: 'SS', showSeason: true

Only Weekday

format: 'WW', showWeekDays: true

Pahela Boishakh (Bengali New Year)

bongabdo('2024-04-14') // 1st Boishakh, 1431
